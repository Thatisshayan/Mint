# MINT — AI Content Studio Frontend

Premium frontend starter for **MINT**, an AI content studio with the tagline:

> Create at the speed of light.

## Included

- React + TypeScript + Vite source code
- Landing page
- Studio dashboard
- Generated content cards
- Library
- Queue
- Calendar
- Mobile previews
- Auth, pricing, and settings sections
- Dark premium mint/teal brand styling
- Framer Motion hero/platform animation
- Fully responsive CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

This is a frontend-only mock implementation with mock data. Connect real auth, database, AI generation, image generation, and publishing APIs when ready.
